How Community Editing Works

  1. Each review page has a "Community Editing" section
  2. Click the Google Drive link for that specific review
  3. Document opens in suggestion mode
  4. Add comments, suggestions, or corrections
  5. Original authors review and approve changes
  6. Approved changes are integrated into the website

Collaboration Guidelines

  • Focus on scientific accuracy and clarity
  • Provide credible sources for new information
  • Respect academic citation standards
  • Maintain professional and constructive tone
  • Suggest specific improvements rather than general criticism

Configuration Files

.htaccess

RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

# Clean URLs
RewriteRule ^reviews/([^/]+)/?$ reviews/$1.html [L]

# Cache optimization
<IfModule mod_expires.c>
    ExpiresActive On
    ExpiresByType text/html "access plus 1 hour"
    ExpiresByType text/css "access plus 1 month"
    ExpiresByType image/png "access plus 1 year"
</IfModule>
