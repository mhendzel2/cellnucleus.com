Video Script Generation Prompts

Basic Educational Video Script

PROMPT FOR GEMINI:

Create a 3-4 minute YouTube video script explaining [NUCLEAR BIOLOGY TOPIC] for an audience of undergraduate biology students and researchers. Structure the script with:

  • Hook (0-15 seconds): Start with an intriguing question or surprising fact
  • Overview (15-30 seconds): Brief topic introduction with key learning objectives
  • Main Content (30 seconds - 3 minutes): Break into 3-4 digestible segments with visual cues
  • Conclusion (3-4 minutes): Summary and call-to-action to visit cellnucleus.com

Include: Quantitative data from research, visual transition cues, engagement questions, and pronunciation guides for technical terms. Tone: Professional but accessible, enthusiastic about the science.

Example topics: "Nuclear Lamina Structure", "Chromatin Dynamics", "DNA Repair Mechanisms"

Scientific Debate/Comparison Script

PROMPT FOR GEMINI:

Create a 5-6 minute "Nuclear Biology Debates" episode script comparing different theories or mechanisms about [TOPIC]. Format as:

  • Introduction: Present the scientific question/controversy
  • Theory A: Explain first perspective with supporting evidence
  • Theory B: Present alternative view with its evidence
  • Current Evidence: Recent studies and what they suggest
  • Expert Perspectives: Quote 2-3 leading researchers

Include specific studies, quantitative comparisons, and methodological differences. Make it feel like a scientific discussion between colleagues.
